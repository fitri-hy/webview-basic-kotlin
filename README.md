# Webview Kotlin

Basic Webview Application Using Kotlin - Android Studio SDK 34
 
## Instalation
- Clone Repositories
```
git clone https://github.com/fitri-hy/webview-basic-kotlin.git
```
- Import Project in Android Studio
- Run / Build Project

## Supported Grandle

compileSdk = 34
minSdk = 24
targetSdk = 34
